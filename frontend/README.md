⚙️ Panduan Instalasi & Pengoperasian Sistem
Untuk menjalankan AssetOps di lingkungan lokal, mulailah dengan memastikan layanan MySQL Anda telah aktif dan buatlah database baru dengan menjalankan perintah CREATE DATABASE revist_inventory; pada terminal SQL Anda. Setelah database siap, buka terminal utama pada folder root proyek (folder paling luar) dan jalankan perintah npm install untuk memasang seluruh modul backend yang diperlukan, kemudian aktifkan server API dengan mengetikkan node server.js hingga muncul notifikasi koneksi berhasil.

Langkah selanjutnya, buka terminal baru (split terminal) dan masuk ke direktori frontend dengan perintah cd frontend. Di dalam folder ini, jalankan kembali perintah npm install untuk memasang dependensi antarmuka, lalu luncurkan dashboard utama dengan perintah npm start. Sistem secara otomatis akan membuka browser dan mengarahkan Anda ke alamat http://localhost:3000. Pastikan kedua terminal (backend dan frontend) tetap berjalan secara bersamaan agar sinkronisasi data analitik dan fitur kontrol stok berfungsi dengan maksimal.
